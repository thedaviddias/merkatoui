# Introduction

Merkato UI is a collection of beautifully designed, accessible, and customizable React components built with Tailwind CSS. Our components are designed to help you build modern web applications faster and with better user experience.

## Getting Started

Install Merkato UI using your preferred package manager:

```bash
# Using pnpm
pnpm add @merkatoui/components

# Using npm
npm install @merkatoui/components

# Using yarn
yarn add @merkatoui/components
```
