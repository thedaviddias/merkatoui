# Introduction

Merkato UI is a collection of beautifully designed, accessible, and customizable React components built with Tailwind CSS and Shadcn. Our components are designed to help you build modern web applications faster and with better user experience.

## Why Merkato UI?

In today's web development landscape, creating consistent, accessible, and beautiful UIs can be time-consuming. Merkato UI solves this by providing:

- **Ready-to-use components** that maintain consistency across your application
- **No vendor lock-in** - components are built with standard React and Tailwind CSS
- **Full customization** - easily adapt components to match your brand identity
- **Developer experience** - strong TypeScript support and intuitive APIs

## Philosophy

We believe that great UI libraries should be both powerful and flexible. Merkato UI is built on these core principles:

- **Design First**: Each component is carefully designed with modern aesthetics and user experience in mind
- **Developer Friendly**: Easy to integrate, customize, and extend
- **Accessibility Matters**: Built following WAI-ARIA guidelines to ensure your applications are accessible to everyone
- **Performance Focused**: Lightweight and optimized for production use
